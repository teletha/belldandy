Cron

Represents a cron expression and provides methods to calculate the next execution time. This class supports both 5-field (minute, hour, day of month, month, day of week) and 6-field (second, minute, hour, day of month, month, day of week) cron expressions.

Cron(String expr)

String expr

The cron expression string.

Constructs a new Cron instance based on the given cron expression.

next(ZonedDateTime base)ZonedDateTime

ZonedDateTime base

The base time to start the search from.
ZonedDateTime

The next execution time as a ZonedDateTime.

Calculates the next execution time after the given base time. This method searches for the next execution time within the next four years.

next(ZonedDateTime base, ZonedDateTime limit)ZonedDateTime

ZonedDateTime base

The base time to start the search from.
ZonedDateTime limit

The upper limit for the search.
ZonedDateTime

The next execution time as a ZonedDateTime.

Calculates the next execution time after the given base time and before the given limit.
